AzureのFaceAPIを使った出退勤アプリのCloudrunでのデプロイ用

<概要>
出退勤を選択し顔判定で人物を特定。DBに情報を保存し
DBの情報を50件表示する。

<使い方>
１．最初の画面から出勤か退勤のボタンを選択
２．内蔵カメラから動画を取得しているのでそれを許可し画面左の赤枠に顔をちょうど収めるようにする。
３．シャッターボタンを押す
４．少し待つと下の方に判定結果が出るので間違っていなければokボタンを押す
５．判定結果とDBの情報50件が表示される

<環境>
・顔の判定はAzureのFaceAPI
・DBはGoogleのSQLを使用
・キーの暗号化にSecretManagerを使用
・デプロイはGCPのCloudrunを使用
